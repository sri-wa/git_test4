  • belajar clone

    • Salah satu istilah dalam Git yaitu git clone. Jika diartikan secara harfiah, istilah clone atau kloning mungkin lebih dekat dengan dunia biologi, sebagai sebutan untuk turunan dari suatu sel yang bereproduksi. Namun pada git, clone berfungsi menyalin file dari suatu repository GitHub ke dalam komputer kita.
